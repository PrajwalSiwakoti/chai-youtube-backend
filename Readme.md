# Chai YouTube Backend

This repository contains the backend code for the Chai YouTube project.
[Model Link] (https://app.eraser.io/workspace/YtPqZ1VogxGy1jzIDkzj)

## Features

- RESTful API for YouTube-like functionalities
- User authentication and authorization
- Video upload and streaming
- Commenting and likes system

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm or yarn
- MongoDB

### Installation

```bash
git clone https://github.com/yourusername/chai-youtube-backend.git
cd chai-youtube-backend
npm install
```

### Running the Server

```bash
npm start
```
